✨ Overall Goal

Build a production-quality Flutter application that looks like a real AI Interview Coach available on the Google Play Store. The app should feature a clean white and emerald green theme, premium UI/UX, smooth animations, responsive layouts, reusable widgets, proper state management, and seamless integration with the existing backend APIs. Every screen should be fully functional, polished, and connected through intuitive navigation to deliver a realistic mobile experience.

⭐ One final suggestion

If your professor is mainly evaluating Flutter, UI/UX, and mobile app development, this 7-screen structure is ideal. It demonstrates routing, state management, API integration, responsive design, and a professional user experience without making the project unnecessarily large. It strikes a good balance between completeness and quality.
